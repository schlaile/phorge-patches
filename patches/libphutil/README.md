# libphutil Patches

Curated patch series against the standalone `libphutil` repository.

This directory is not the default target for current upstream-oriented work.
For active code, check `phorge-arcanist` first: much of the former
`libphutil`-level functionality lives there now.

Recommendation:

- keep patches small and thematic
- prefer neutral, upstream-usable commit messages in the exported patches

## Current Scope

- keep only patches here which truly target standalone `libphutil`
- if a fix turns out to belong in `phorge-arcanist`, remove it from this
  directory and curate it there instead
